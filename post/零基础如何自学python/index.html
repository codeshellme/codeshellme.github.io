1，什么是Python

人与人的交流需要人类语言，人与计算机交流（控制计算机）需要计算机语言。

Python 是众多计算机语言中的一种，每种计算机语言都有它自己的优势。

对于初学者来说，Python 是最容易上手，也是一门功能强大的计算机语言。

2，为什么要学Python

对于专业人士，也就是将来想从事编码工作的人（程序员）来说，与其它语言相比，Python 语言学习起来更加容易，没有底层晦涩难懂的概念。学会了Python，再去学其它语言，就更加容易了。

对于非专业人士，也就是不从事编码工作的人来说，Python 语言可以做数据分析，可以处理Excel 等办公软件，也可以方便你处理日常的工作。

3，准备工作

需要的设备：

首先你需要一台电脑（并不需要多么强悍的配置），不管是Windows 还是MacOs，只要你用起来顺手，并且能上网即可。

当然也有一些手机软件可以运行Python 语言，但是毕竟手机没有电脑方便。

如果想学Python，一台电脑，一根网线还是必不可少的。

需要的资料：

学习新知识，每个人都有不同的习惯，可以通过纸质书籍，在线教程，视频网站均可。

4，Python 书籍推荐

以下4 本书，任选一本即可。这些书都有PDF 版，可以先从网上下载电子版，如果觉得不错可以再买纸质版。

也可以从网上查一下评价，一般豆瓣8.0 分以上的书，都还是不错的。至于书籍的版本，可以买最新版的。

  • 《python 编程快速上手》豆瓣9.0 分（入门）
  • 《Python 编程：从入门到实战》豆瓣9.1 分（入门）
  • 《流畅的Python》豆瓣9.4 分（进阶）
  • 《Python Cookbook》豆瓣9.3 分（进阶）

5，Python 在线教程

以下教程百度搜索，即可找到。

  • 《Python 简明教程》沈洁元（译），本教程强烈推荐，简洁明了。
  • 《Python 菜鸟教程》菜鸟教程不仅可以学习Python 语言，还可以学习其它编程相关的知识。
  • 《廖雪峰 Python 教程》廖雪峰老师写的教程简单易懂，很适合初学者。

6，Python 视频资源

  • 慕课网：这个网站有文字教程也有视频教程，有免费的，也有收费的，大家可以自取所需。
  • 传智播客：这是一个专业的IT 教育机构，其官网有很多不错的免费视频教程。
  • B 站：这个网站大部分年轻人都应该知道，也有很多不错的视频教程，可以选择播放量较高的进行学习。
  • 极客时间：这是一个专业的IT 教育付费APP，有视频也有文字课程，都是付费教程，不过质量还是可以的。
